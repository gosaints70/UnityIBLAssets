UnityIBLAssets
==============

UnityでリアルタイムIBLをしてみた時のアセット群です。

このアセット群の使用は非営利での使用に限ります。

使用方法などはこちらのページを参照ください。

http://kosakasakas.github.io/blog/2014/10/18/unity-realtime-ibl/
